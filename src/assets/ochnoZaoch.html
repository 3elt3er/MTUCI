Очно-заочный бакалавриат предполагает обучение по 1 направлению. Обучение проходит по вечерам 3-4 раза в неделю.


Профиль: Организация и технология защиты информации

Проходной балл 2022:-

Количество мест: Б 0/ П 100

Стоимость 2023: 45 000 рублей за семестр

О направлении:

Образовательная программа «Организация и технологии защиты информации» осуществляет подготовку специалистов в области информационных технологий.

В рамках данного профиля студенты обучаются по двум основным направлениям обучения:

программирование ( 1 курс с++, дальше Java, python) Есть дисциплины связанные с искусственным интеллектом и машинным обучением

информационная безопасность - изучаются различные аспекты данной тематики, от физической защиты объектов информатизации до защиты данных при работе с информационными системами.

Профессии: IT-архитектор Аналитик IT-менеджер IT-консультант Программист Проектировщик систем умный дом, умный город Специалист тех. поддержки Разработчик ПО для систем управления Разработчик интеллектуальных систем управления Системный разработчик Embedded Linux для умных устройств Инженер по БПЛА Инженер по разработке алгоритмов управления БПЛА Специалист по внедрению цифровых решений для ключевых клиентов Инженер-программист АСУ ТП
